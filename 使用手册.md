# ChubbSafes 竞品监测 · 使用手册（市场团队版）

本系统帮助市场团队自动收集、对比竞品保险柜的价格、规格、促销与口碑。全部操作都在
**网页看板**里完成，无需写代码。

---

## 一、日常使用（每天）

1. 双击项目根目录的 **`启动看板.bat`**。
2. 稍等几秒，浏览器会自动打开看板（网址 `http://127.0.0.1:8010`）。
3. 用完后，**关闭那个黑色命令行窗口**即可停止看板。

> 想更新到最新数据？在看板右上角点 **「运行抓取」**，或双击 **`每日更新数据.bat`**。

---

## 二、看板各页面

| 页面 | 作用 |
|------|------|
| **仪表盘** | 今日总览：竞争对手数、在监产品数、今日变化、数据渠道。 |
| **每日报告** | AI 生成的中文竞品动态摘要 + 价格异动 + 新品。 |
| **竞争对手** | 各品牌档案；可标记「重点关注」。 |
| **产品情报** | 所有竞品产品表：可**搜索/排序/编辑/删除**，可点「新增产品」手动录入，点产品名看详情。 |
| **多平台比价** | 同一型号在京东/天猫/苏宁的价格并排对比。 |
| **对标分析** | 选一款集宝产品 + 一款竞品，查看每项指标的**完整计算过程**（价格差/容积差/元升/性价比等）。 |
| **市场情况** | 市场地图 + 价格变动 + 市场趋势（图表）。 |
| **研究智能体** | 四个 AI 助手（见下）。 |
| **设置/系统状态** | 检查各密钥是否正常、数据覆盖情况。 |

---

## 三、AI 研究智能体（研究智能体页面）

进入后有四个卡片，各有独立页面和详细说明：

- **品牌深挖**：输入一个竞品品牌，AI 联网检索并核查后把可信信息入库。
- **竞品信息自动化搜集**：给一个产品 ID，AI 浏览它的链接补全缺失参数。
- **文档摄取**：上传竞品分析 PPT / 产品手册 PDF，自动抽取产品表。
- **舆情分析**：搜索并分析「集宝 ChubbSafes」（或任意品牌）的网络口碑，生成舆情报告。

> AI 找到的存疑信息不会直接入库，会进入「信息确认队列」等你人工 **采纳/驳回**。

---

## 四、其他录入方式

- **手动录入产品**：产品情报页 →「新增产品」，可粘贴商品文字让 AI 识别，或直接填表 + 上传图片。
- **竞品促销邮件**：用专用邮箱订阅竞品 newsletter/店铺关注邮件后，运行
  `py -m uv run chubb-ci ingest-email` 即可把促销信息入库（邮箱在 `.env` 配置）。

---

## 五、换新电脑 / 首次安装

1. 把整个项目文件夹拷到新电脑。
2. 双击 **`首次安装.bat`**，按提示完成（会自动装依赖与浏览器内核）。
3. 确认根目录 **`.env`** 文件里的各项密钥已填好（DeepSeek、邮箱授权码、Bocha 搜索、京东联盟等）。
4. 之后每天用 `启动看板.bat` 即可。

---

## 六、遇到问题

- **浏览器打不开页面**：稍等 5–10 秒待服务启动，然后刷新 `http://127.0.0.1:8010`。
- **端口被占用 / 打不开**：关闭旧的命令行窗口再重开一次 `启动看板.bat`。
- **某项功能提示「未配置」**：到「设置/系统状态」页查看，按提示在 `.env` 补齐对应密钥。
- **`.env` 密钥务必保密**，不要发到微信/邮件/GitHub。
